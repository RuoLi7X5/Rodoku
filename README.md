# 数独秩理论研究（Rank Sudoku）

只研究 **9×9 标准数独**。本项目目标是：用 **Truth/Link/Rank** 的“秩结构”统一表达并驱动删数推理，并将结构可视化呈现出来。

## 运行

```bash
npm install
npm run dev
```

## 已实现能力（阶段性）

- **盘面与规则**：行/列/宫不重复校验；提示数不可编辑；用户可填数/清空。
- **候选系统**：
  - 候选数以 **3×3** 布局显示在格子内。
  - `forbidden` 持久删候选：被删的候选不会被“全标/刷新候选”加回。
  - 填入数字后会自动对同行/同列/同宫做候选排除（只删不加）。
- **导入（兼容多格式）**：
  - **81格格式**：支持 `0` / `.` / 空格表示空格（长度=81 时空格视为格子；否则空格视为分隔符）。
  - **特殊格式**（示例）：
    `:...:3.+12.+9.....+9...:724 627 ... ::`
    - `.` 空格
    - `+数字` 表示用户填数
    - 无 `+` 的数字表示题面提示数
    - `724` 表示删除候选：删 `r2c4` 的 `7`
  - 导入成功后会 **默认全标候选**（基于行列宫排除，且尊重 forbidden）。
- **前端秒解**：
  - 导入题目后自动求解：
    - **无解**：拒绝导入
    - **多解**：允许导入并提示（展示其中一个解作为导航）
    - **唯一解**：用于导航
  - 解以 9×9 缩略图显示在盘面左侧空白区。
- **秩结构查询与可视化**：
  - Truth 数量范围默认 **1–8**，支持查询/Stop 中断。
  - 仅计算 **R < 3（R=0/1/2）** 的结构。
  - 结构可视化：
    - Truth（强区域）：粗实线连接候选，并圈出被涵盖候选点
    - Link（弱区域）：单虚线显示（仅呈现强区域集合里的候选点）
    - 删数：红色实心圈标记
  - 列表：
    - 按显示顺序编号
    - 单行摘要：`1. 3T4L r23456c4<>8`
    - 同删数结构自动去重
    - 点击“应用”后会把删数落盘，列表会自动隐藏已失效/同删数项（无需重新查询）

## 基础交互

- **点选候选**：点击候选数字，会出现浅棕色框反馈。
- **右键填数/取消**：在同一格上右键：
  - 空格 → 填入该候选数字
  - 若该格已填且等于该候选 → 再次右键清空
